# kvelf
kvelf is an ELF parser.


## Installation

1. Clone the repo
   ```sh
   git clone https://github.com/d4rk3agle/kvelf
   ```
2. Make install script executable 
   ```sh
   chmod +x ./install.sh
   ```
3. Install
   ```sh
   sudo ./install.sh
   ```
